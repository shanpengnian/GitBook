# SXPCWL-CMS 简介

SXPCWL-CMS管理系统，是基于JAVA 的综合管理系统,安全、稳定、灵活,全国用户超过万,致力于为企业、学校、政府、商铺、酒店、餐厅。

项目应用包含：运营系统、App、小程序、网站、公众号建设，提供企业级定制化管理系统解决方案。基于java后台语言，已功能闭环，且达到商用标准体系。
